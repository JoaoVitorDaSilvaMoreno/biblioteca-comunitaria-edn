# 📚 Biblioteca Comunitária EDN

Projeto Final desenvolvido para o curso **Tecnologia para Iniciantes** da **Escola da Nuvem (EDN)**.

## 📖 Sobre o Projeto

A Biblioteca Comunitária EDN é um sistema web desenvolvido em Python utilizando Flask, criado para auxiliar no gerenciamento de uma biblioteca comunitária.

O sistema permite:

- Cadastro de livros
- Cadastro de usuários
- Empréstimo de livros
- Devolução de livros
- Pesquisa de livros
- Histórico de movimentações
- Login administrativo
- Feedback por voz utilizando o PowerShell Speech Synthesizer

---

## 🚀 Tecnologias Utilizadas

- Python
- Flask
- HTML5
- CSS3
- Jinja2
- PowerShell Speech API

---

## 🎯 Funcionalidades

### 📚 Livros
- Cadastro de novos livros
- Visualização do acervo

### 👤 Usuários
- Cadastro de usuários
- Consulta de usuários cadastrados

### 📖 Empréstimos
- Registro de empréstimos
- Controle de disponibilidade dos livros

### ↩️ Devoluções
- Registro de devolução de livros

### 🔍 Pesquisa
- Busca de livros por título

### 📋 Histórico
- Registro das movimentações realizadas no sistema

### 🔐 Login Administrativo
- Controle de acesso ao sistema

## 🖥️ Como Executar Localmente

### 1. Clone o repositório

```bash
git clone https://github.com/SEU-USUARIO/biblioteca-comunitaria-edn.git
```

### 2. Entre na pasta do projeto

```bash
cd biblioteca-comunitaria-edn
```

### 3. Instale as dependências

```bash
pip install flask
```

### 4. Execute o sistema

```bash
python app.py
```

### 5. Acesse no navegador

```text
http://127.0.0.1:5000
```
